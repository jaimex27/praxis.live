audio:filter:comb

A comb filter.

Ports

ID Type Direction Values Description
in audio in Audio input.
out audio out Audio output.
frequency control in number(20..20000) Frequency of filter.
feedback control in number(0..1) Amount of signal fed back into input.
mix control in number(0..1) Mix.
link link bidi Link comb filter components together (all parameters will be synced).
