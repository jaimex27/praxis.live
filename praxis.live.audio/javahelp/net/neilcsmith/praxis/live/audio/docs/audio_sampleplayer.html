audio:sampleplayer

A mono sample player. Supports looping, in/out points, variable speed.

Samples are loaded in the background. If using the sample port to load new samples, make use of the ready and error ports to track loading.

The player will not start automatically. To get it to start when the audio root is started, use a core:start-trigger attached to the play port

Ports

ID Type Direction Values Description
out audio out Audio output.
sample control in resource The audio file to load.
position control in number(0..1) Position within audio sample.
start control in number(0..1) Start point for playback / looping.
end control in number(0..1) End point for playback / looping.
speed control in number(-4..4) Playback speed.
play control in any Trigger playback.
stop control in any Stop playback.
ready control out {} New sample loaded.
error control out {} Error loading new sample.

Controls
