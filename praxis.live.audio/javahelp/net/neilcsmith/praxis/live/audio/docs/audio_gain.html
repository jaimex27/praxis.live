audio:gain

Control audio gain (volume). Multiple gain components can be controlled as one using the link port.

Ports

ID Type Direction Values Description
in audio in Audio input.
out audio out Audio output.
level control in number(0..2) Level of gain.
link link bidi Link gain components together (the level parameter will be synced).

Controls

ID Type Input Output Default Description
level property number(0..2) 1 Level of gain.
