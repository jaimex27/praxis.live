audio:filter:iir

An Infinite impulse response (IIR) filter with various types.

Types are LP6, LP12, HP12, BP12, NP12, LP24, HP24.

Ports

ID Type Direction Values Description
in audio in Audio input.
out audio out Audio output.
frequency control in number Frequency of filter.
resonance control in number(0..30) Resonance of filter.
mix control in number(0..1) Mix.
link link bidi Link IIR filter components together (all parameters will be synced).

Controls

ID Type Input Output Default Description
frequency property number 0 Frequency of filter.
resonance property number(0..30) 0 Resonance of filter.
mix property number(0..1) 0 Mix.
