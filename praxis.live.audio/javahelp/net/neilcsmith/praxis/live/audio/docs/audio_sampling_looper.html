audio:sampling:looper

A mono phrase sampler. Supports in/out points, variable speed.

Ports

ID Type Direction Values Description
in audio in Audio input.
out audio out Audio output.
position control in number(0..1) Position within audio sample.
start control in number(0..1) Start point for playback / looping.
end control in number(0..1) End point for playback / looping.
speed control in number(-4..4) Playback speed.
play control in any Trigger playback.
stop control in any Stop playback.
record control in any Start recording.

Controls

ID Type Input Output Default Description
loop-size property number(0..60) 1 The loop size (in seconds).
position property number(0..1) 0 Position within audio sample.
start property number(0..1) 0 Start point for playback / looping.
end property number(0..1) 0 End point for playback / looping.
range property number(0..1) number(0..1) 0 1 Specify the start and end loop points in one go.
This property doesn't currently show up in Praxis LIVE, but can be bound to using a gui:h-rangeslider.
speed property number(-4..4) 0 Playback speed.
loop property boolean false Loop playback.
play action Trigger playback.
stop action Stop playback.
record action Start recording.
