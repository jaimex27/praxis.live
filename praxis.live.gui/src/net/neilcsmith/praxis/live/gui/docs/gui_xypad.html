gui:xy-pad

An XY pad. Different controls can be bound to vertical and horizontal movement.

If minimum, maximum or scale value are not set, the pad will attempt to configure these from the bound controls. If these are not specified by the control they will default to 0, 1 and Linear respectively.

Controls

ID Type Input Output Default Description
label property string {} The component label.
binding-x property control address {} Address of the control to bind to for horizontal movement.
binding-y property control address {} Address of the control to bind to for vertical movement.

minimum-y property number(0..1) or {} {} Minimum value for vertical movement (see note above).
maximum-x property number(0..1) or {} {} Maximum value for horizontal movement (see note above).
maximum-y property number(0..1) or {} {} Maximum value for vertical movement (see note above).
scale-x property string(Linear, Exponential) or {} {} Scale type for horizontal movement (see note above).
scale-y property string(Linear, Exponential) or {} {} Scale type for vertical movement (see note above).
