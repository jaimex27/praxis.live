gui:button

A button. Can be bound to a specific control or execute a script when clicked.

Controls

ID Type Input Output Default Description
label property string {} The button label.
binding property control address {} Address of the control to bind to.
values property array {} Array of values sent as arguments to the bound control when the button is clicked.
on-click property string(text/x-praxis-script) {} A script to be run when the button is clicked. Can be used instead of or as well as the control binding.
layout property string {} A layout string (based on the format of MigLayout).
