gui:h-slider

A horizontal slider.

If a minimum, maximum or scale value are not set, the slider will attempt to configure these from the bound control. If these are not specified by the control they will default to 0, 1 and Linear respectively.

Controls

ID Type Input Output Default Description
label property string {} The component label.
binding property control address {} Address of the control to bind to.
minimum property number(0..1) or {} {} Minimum value (see note above).
maximum property number(0..1) or {} {} Maximum value (see note above).
scale property string(Linear, Exponential) or {} {} Scale type (see note above).
layout property string {} A layout string (based on the format of MigLayout).
