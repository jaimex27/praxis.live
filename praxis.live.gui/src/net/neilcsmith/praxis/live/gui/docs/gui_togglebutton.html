gui:togglebutton

A toggle button.

When syncing to a bound control, the value of on-value is used to determine whether the button appears selected.

Controls

ID Type Input Output Default Description
label property string {} The button label.
binding property control address {} Address of the control to bind to.
on-value property any true Value sent to the bound control when the button is selected. This value is also used for syncing.
off-value property any true Value sent to the bound control when the button is de-selected.
layout property string {} A layout string (based on the format of MigLayout).
