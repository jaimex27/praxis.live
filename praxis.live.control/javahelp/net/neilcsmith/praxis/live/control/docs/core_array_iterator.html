core:array:iterator

Loops through an array of arguments, jumping to the next each time it is triggered. Can be set to jump a (random) number of elements ahead, determined by min-skip and max-skip. Can loop continuously forwards, or ping-pong back and forth through the array.

If the array is empty, an empty string will be sent instead.

Ports

ID Type Direction Values Description
trigger control in argument Triggers sending of a value when a signal is received.
out control out argument Outputted value.
values control in array The array of values.

Controls

ID Type Input Output Default Description
trigger action Trigger sending of value from output port.
values property array {} The array of values.
min-skip property number 1 Minimum number of elements to jump when triggered.
max-skip property number 1 Maximum number of elements to jump when triggered.
ping-pong property boolean false Loop back and forth.
