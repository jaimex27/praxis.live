core:array:random

Stores an array (list) of arguments, and selects and sends one randomly when triggered.

If the array is empty, an empty string will be sent instead.

Ports

ID Type Direction Values Description
trigger control in argument Triggers sending of a value when a signal is received.
out control out argument Outputted value.
values control in array The array of values.

Controls

ID Type Input Output Default Description
trigger action Trigger sending of value from output port.
values property array {} The array of values.
