core:array:random

Stores an array of arguments, and selects and sends one randomly when triggered. If the array is empty, an empty string will be sent instead.

Ports

trigger
control in | argument | Triggers sending of a value when a signal is received.
out
control out | argument | Sends selected value when triggered.

Controls

values
property | array | {} | Array of possible values.
ID
trigger | - | - | Trigger sending of value from output port.
