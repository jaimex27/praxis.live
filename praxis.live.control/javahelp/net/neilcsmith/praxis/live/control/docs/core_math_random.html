core:math:random

Sends a random number between minimum and minimum + range when triggered. If range is zero, the value of minimum will be sent when triggered.

Ports

minimum
control in | number | Value of minimum.
range
control in | number(>=0) | Value of range.
trigger
control in | argument | Trigger random number to be sent from output.
out
control out | number | Output.

Controls

minimum
property | number | 0 | Value of minimum.
range
property | number(>=0) | 1 | Value of range.
trigger
function | - | - | Trigger random number to be sent from output.
