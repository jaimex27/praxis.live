core:timing:animator

Animate a value.

Ports

ID Type Direction Values Description
to control in number Start animating to value.
out control out number Animated value.
value control in number Set current value. Stops animation.
time control in number(0..300) Time (in seconds) of animation.

Controls

ID Type Input Output Default Description
to function number Start animating to value.
value property number 0 Current value. Set stops animation.
time property number(0..300) 0 Time (in seconds) of animation.
