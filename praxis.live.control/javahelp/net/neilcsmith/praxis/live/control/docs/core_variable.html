core:variable

Stores a variable and sends it whenever triggered. Unlike core:property it does not send on each update.

Ports

value
control in | argument | Sets value to signal received.
trigger
control in | argument | Triggers sending of value when a signal is received.
out
control out | argument | Sends value when triggered.

Controls

value
property | argument | {} | Value of this component.
trigger
function | - | - | Trigger sending of value from output port.
