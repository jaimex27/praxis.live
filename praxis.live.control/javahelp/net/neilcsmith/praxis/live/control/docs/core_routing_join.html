core:routing:join

Join two input signals. A signal to either input port will arm the component. A signal to the other input port will then pass through, and un-arm the component.

Repeated signals to the arming port will be ignored.

Ports

ID Type Direction Values Description
in-1 control in any First input port.
in-2 control in any Second input port.
out control out Output port.
reset control in any Un-arm component.

Controls

ID Type Input Output Default Description
reset action any Un-arm the component.
