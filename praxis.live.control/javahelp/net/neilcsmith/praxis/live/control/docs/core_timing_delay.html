core:timing:delay

Delay an input signal for a period of time.

A delay of 0 will delay the signal until the next control period (video frame, audio buffer, etc.) is processed.

Ports

ID Type Direction Values Description
in control in any Input signal.
out control out any Output signal.
time control in number(0..3600) Delay time in seconds.

Controls

ID Type Input Output Default Description
time property number(0..3600) 0 Delay time in seconds.
