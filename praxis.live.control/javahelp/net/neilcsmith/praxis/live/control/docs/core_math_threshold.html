core:math:threshold

Takes a number on its input port and sends it from its out-high port if greater than or equal to the threshold value, or from the out-low port if below the threshold value.

Ports

ID Type Direction Values Description
in control in number Input value.
out-low control out number Value if lower than threshold.
out-high control out number Value if higher than (or equal to) threshold.
threshold control in number Value of threshold.

Controls

ID Type Input Output Default Description
threshold property number 0 Value of threshold.
