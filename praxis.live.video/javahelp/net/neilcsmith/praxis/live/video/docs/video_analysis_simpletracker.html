video:analysis:simple-tracker

A simple blob tracker. Tracks the largest blob in an image.

The input video should be high-contrast monochrome. This can be achieved using video:analysis:difference set to Threshold mode. It may also be good to pass the output of the difference component through video:filter:blur.

Ports

ID Type Direction Values Description
in video in Video input.
out video out Video output.
x control out number(0..1) The horizontal centre point of the blob.
y control out number(0..1) The vertical centre point of the blob.
width control out number(0..1) The width of the blob.
height control out number(0..1) The height of the blob.

Controls

ID Type Input Output Default Description
debug property boolean false Draw bounding rectangles of detected blobs on the image. The tracked blob is highlighted.
