video:output

If in the first level underneath a root that can render video, this component will provide output to the screen. If further down the hierarchy, it will provide output from its container.

Ports

in
video in | Video signal to be output.

Controls

width
property | number(>=1) | {} | If set this scales the video output.
It does not change the size of the actual images in the graph. To change this, use the width & height properties on video:root
height
property | number(>=1) | {} | As above.
rotation
property | number(0,90,180,270) | {} | If set, this controls the rotation of the output image.
