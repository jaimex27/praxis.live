video:player

Play a video file. It is also possible to pause and move through the video file using the position property - for this purpose a video codec such as MJPEG is recommended to keep seek times to a minimum.

The player will not start automatically. To get it to start when the video root is started, use a core:start-trigger attached to the play port (or the pause port).

Ports

ID Type Direction Values Description
out video out Video output.
position control in number(0..1) Change position within video file.
play control in any Start playback.
pause control in any Pause playback.
stop control in any Stop playback.

Controls

ID Type Input Output Default Description
video property resource {} The video file to play.
position property number(0..1) 0 Position within the video file.
play action Start playback.
pause action Pause playback.
stop action Stop playback.
