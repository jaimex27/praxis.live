video:opengl:filter

A filter that allows live-coding video effects using the OpenGL Shading Language (GLSL). Currently only allows to set the fragment shader.

For obvious reasons, this component only works correctly when using the OpenGL renderer. The default renderer can be set in Tools / Options / Video.

Ports

ID Type Direction Values Description
in video in Video input.
out video out Video output.
u1 - u8 control in number(0..1) Parameters that can be read from the shader. Use uniform float u1, etc. to access values.

Controls

ID Type Input Output Default Description
fragment property string {} A GLSL fragment shader. NB. Due to the nature of the GLSL compiler, this control does not currently return an error if there's a problem with the shader code. Check the log (View / IDE Log) for messages if the shader does not seem to work.
u1 -u8 property number(0..1) 0 Parameters that can be read from the shader. Use uniform float u1, etc. to access values.
