video:snapshot

This component provides the ability to capture and display a still frame from its video input. It extends this ability with the option to fade from the previous captured image to the new one over a period of time, and to mix the new image with the previous one.

Ports

ID Type Direction Values Description
in video in Video signal to capture from.
out video out Video output.
fade-time control in number(0..3600) Period in seconds to fade from previous image to current image.
mix control in number(0..1) Level to mix current image with previous image.
trigger control in any Trigger new snapshot.
reset control in any Clear snapshot.

Controls

ID Type Input Output Default Description

mix property number(0..1) 1 Level to mix current image with previous image.
