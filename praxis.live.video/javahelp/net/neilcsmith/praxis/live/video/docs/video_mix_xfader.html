video:mix:xfader

Cross fade between two video signals, using the blend mode set.

Available blend modes are Normal, Add, Difference, BitXor

Ports

ID Type Direction Values Description
in-1 video in First video input.
in-2 video in Second video input.
out video out Video output.
mode control in string(from blend modes) Blend mode to use during cross fade.
mix control in number(0..1) Mix amount (0 all in-1, 1 all in-2

Controls

ID Type Input Output Default Description
force-alpha property boolean false Force src to have an alpha channel, even if output doesn't (defaults to same as output).
mode property string(from blend modes) Normal Blend mode to use during cross fade.
mix property number(0..1) 1 Mix amount (0 all in-1, 1 all in-2
